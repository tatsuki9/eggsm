## Installation
```
go get github.com/tatsuki9/eggsm/cmd/eggsm
```

## Usage
```
go run main.go -profile <profile name> -secret_id <secret manager name>
```

## Example
```
$ go run main.go -profile ex_profile -secret_id ex_secret
DB_NAME=hoge
DB_PASSWORD=hoge
DB_HOST=hoge
DB_PORT=hoge
DB_USERNAME=hoge
```
